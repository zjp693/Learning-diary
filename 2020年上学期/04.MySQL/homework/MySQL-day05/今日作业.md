# 建表语句

```sql
 CREATE TABLE EMP 

(EMPNO FLOAT(4) PRIMARY KEY, -- 员工编号

ENAME VARCHAR(10), -- 员工姓名

JOB VARCHAR(9), -- 员工职位

MGR FLOAT(4), -- 员工上级工号

HIREDATE DATE, -- 生日

SAL FLOAT(7,2), -- 薪水

COMM FLOAT(7,2), -- 年终奖

DEPTNO FLOAT(2)  -- 部门号
```
